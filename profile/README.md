# Read By Agents

Citation infrastructure for dated content. Newsletter operators forward issues to `convert@readbyagents.com` and each issue becomes agent-readable markdown at a permanent URL — discoverable via MCP, public REST, and feed endpoints. Free forever. Operators keep attribution and see which agents cite them.
